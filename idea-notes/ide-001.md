# カタルシスとは何か

- 発生の源泉
  - ストレスの解放
  - 切望の成就
  - 報われた
- 感情
  - 万感の思い
  - 感動
  - うれしい
  - 悲しい
  - 強く読者の欲する感情の発生
  - 終焉の感情 => 怒りなどは始まりの感情、落ち着くところに落ち着かないと終われない

- カタルシスとは読者に内在的に欲求のある感情（ポジティブ・ネガティブ双方含む）が（広義の）解決によって一気に訪れること
- 読者の内在的な欲求を高めるシチュエーション
- 高まる欲求と欲求を高める要素
  - 喜悦――「苦難」が報われる
  - 安堵――「危険」から逃れる
  - 納得――「未知」を既知にする
  - 正義――「不正」が公正になる
  - 悲哀――「惨状」から悲劇的に解放される
- 僕がデザインしたい感情
  - 充足感
    - そのために必要な積み重ね => 様々な困難を乗り越える
